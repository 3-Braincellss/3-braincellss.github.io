Welcome to Comp0010 Shell Python P14’s documentation!¶

Note

This shell was designed and developed by team-14-python for the COMP0010-Software engineering coursework, entirely for learning purposes. It is not intended to be practically usable.

Getting Started¶

Installation¶

To install all project dependencies run this from the root directory:

$ python -m pip install -r requirements.txt

Running¶

From the root directory run:

$ python src/run.py

or:

$ bash sh

Here you go! You can now play around with our shell.
