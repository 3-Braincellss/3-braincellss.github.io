Tools¶

prettify_path(path)¶

Prettifies a given path

Converts a given path like this:

/home/mow/Documents/file.txt

Into this:

h/m/D/file.txt

Parameters

path (str) – path to be prettified.

Returns

A prettified path.

Return type

str

simple_globbing(args)¶

Globs the list of given arguments and returns an extended list of globbed paths.

Parameters

args (list) – A list of paths to be globbed.

Returns

A list of globbed paths.

Return type

list

read_from_file(path, app_str)¶

Reads from a file and returns its contents.

Parameters

  • path (str) – The path of the file to be read.

  • app_str (str) – Name of the app that might cause the error.

Returns

File contents

Return type

str

Raises

RunError – If a given path is a directory or a given path doesn’t exist.
