Commands¶

Module Contents¶

This module contains definitions for different types of commands that can be executed in our shell, as well as the common command interface and a command factory for easy object creation.

The list of supported commands:

  • Call - Command primitive which runs a specific app.

  • Pipe - Can redirect output of one command as the input of another

  • Sequence - Runs 2 commands in a sequence

run(inp, out)¶

Handles possible redirections and runs the app.

Can handle both input and output redirections.

In case there is an input redirection and some piped input it will prioritise redirected input.

In case there is an output redirection will create an empty deque for the output and store results in the output file. Will return any output that was passed as the argument.
